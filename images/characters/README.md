## Image Requirements

- Format: JPG, PNG, or WEBP
- Recommended size: 400x500 pixels (portrait orientation)
- Max file size: 1MB per image

## Finding Images

You can find Stranger Things images from:
- Google Images (search "stranger things [character name]")
- Unsplash.com
- Pexels.com
- Netflix promotional images
- Fandom wikis

Make sure to use images you have permission to use!
